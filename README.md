# 📡 Pusat Kendali Kehumasan — BPS Sultra

Dashboard modern berbasis Streamlit untuk monitoring dan penjadwalan terintegrasi lintas channel kehumasan BPS Provinsi Sulawesi Tenggara.

## 🚀 Fitur Utama
*   **Executive Summary**: Ringkasan performa agenda bulanan dan tahunan.
*   **Nexus Slate Design**: UI premium dengan efek glassmorphism dan dark mode.
*   **Kalender Terintegrasi**: Visualisasi jadwal lintas 11 channel sheet.
*   **Optimasi Mobile**: Tata letak responsif untuk akses cepat via smartphone.
*   **Data Real-time**: Koneksi sinkron ke Google Sheets dengan performa tinggi.

## 🛠️ Instalasi Lokal
1.  Clone repository ini.
2.  Install dependencies: `pip install -r requirements.txt`
3.  Simpan credentials Google Sheets di `.streamlit/secrets.toml`.
4.  Jalankan: `streamlit run app.py`

## 📡 Deployment
Didesain untuk Streamlit Cloud. Pastikan data rahasia dimasukkan ke dalam **Secrets Manager** di dashboard deployment.
